# AI生成素材要求

## 优先方案：AI生成动态视频

AI生成素材**优先尝试生成动态视频**，而非静态图片。可用工具按优先级：

1. **WorkRally AI生视频**（需API Key，若已配置则优先使用）
2. **其他AI视频生成工具**（如有可用MCP工具）

若AI视频生成工具可用：
- 生成5-8秒动态视频，直接用于素材区
- 提示词：无文字、无人脸、与文案语义匹配的场景动态画面（场景类型由文案实际内容决定，不限定某一类型）
- 生成后仍需验收12项检查

## 备选方案：AI生成图片 + FFmpeg运镜

**当AI视频生成不可用时**（如WorkRally API Key未配置），使用 `image_generate` 生成静态图片，再通过FFmpeg zoompan做运镜模拟动态效果。**不能直接贴静态图。**

```bash
# zoompan 缓慢放大
ffmpeg -y -i input.jpg -vf "scale=4000:-1,zoompan=z='min(zoom+0.0015,1.5)':d=150:s=1080x768:x='iw/2-(iw/zoom/2)':y='ih/2-(ih/zoom/2)':fps=30" -t 5 output.mp4

# zoompan 左右平移
ffmpeg -y -i input.jpg -vf "scale=4000:-1,zoompan=z=1.2:d=150:s=1080x768:x='if(lte(on,75),0,(iw-1080*1.2)*(on-75)/75)':y='ih/2-(ih/zoom/2)':fps=30" -t 5 output.mp4
```

## 运镜类型选择

| 转场运镜 | 适用场景 | FFmpeg参数 |
|----------|----------|------------|
| zoom-in | 强调细节、聚焦主体 | `zoom+0.0015`缓慢放大 |
| zoom-out | 展示全貌、拉开视野 | `zoom-0.0015`缓慢缩小 |
| pan-left | 展示横向场景（产线、流水线） | x从右到左移动 |
| pan-right | 展示横向场景（反向） | x从左到右移动 |
| tilt-up | 展示纵向场景（高楼、设备） | y从下到上移动 |
| tilt-down | 展示纵向场景（反向） | y从上到下移动 |
| static | 数据可视化动画自带动效 | 不加zoompan |

- 运镜类型根据画面内容和分镜方案中的转场运镜字段选择
- 持续时间匹配TTS时长
- 效果必须丝滑，不能生硬跳帧
- AI图片提示词：无文字、无人脸、与文案语义匹配的场景（场景类型由文案实际内容决定，不限定某一类型）
