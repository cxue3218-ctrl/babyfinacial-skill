## 一、通用规则（所有平台适用）

来源：SKILL.md `0b-extra` 节

### 基本原则

- **必须用广泛词**：用行业大类词搜索，如"半导体"、"芯片制造"、"光纤"、"城市夜景"、"股票 K线 素材"
- **禁止太窄的词**：如"台积电2024Q3涨价公告截图"、"某某UP主第3期视频"——太窄搜不到
- **关键词组合**：1个主题词 + 1-2个修饰词，如"半导体 工厂 航拍"、"光纤 生产 线"
- **备选关键词**：每镜至少准备2组关键词，主关键词搜不到时切换备选

### 后缀"素材"备选

当主关键词搜不到或结果不理想时，在关键词后加"素材"后缀作为备选词：

| 主关键词 | 备选（加"素材"） |
|---------|----------------|
| 半导体 | 半导体素材 |
| 芯片制造 | 芯片制造素材 |
| 光纤 | 光纤素材 |
| 数据中心 | 数据中心素材 |

部分素材网站对带"素材"后缀的搜索词有更好的匹配结果。

## 二、分镜方案关键词填写规则

来源：SKILL.md `0d` 节（分镜方案表）

### 填写方式

分镜方案表中的"搜索关键词"列填写**中文关键词**，遵循通用规则（广泛词 + 组合词）。

示例：

| 镜号 | 素材描述 | 搜索关键词 | 备选方案 |
|------|---------|-----------|---------|
| seg_00 | 工厂航拍/科技园区 | 半导体 工厂 航拍 | 平台实拍:芯片产线 |
| seg_05 | 芯片工厂产线 | 芯片 制造 产线 | 素材网站:半导体工厂 |
| seg_12 | 光纤拉丝工艺特写 | 光纤 生产 拉丝 | 数据可视化:产能对比 |

### 填写要点

- 关键词用**空格分隔**（不是逗号或加号）
- 每个关键词2-4个词
- 备选方案标注平台前缀：`平台实拍:`、`素材网站:`、`数据可视化:`、`AI生成视频`

## 三、素材网站采集关键词策略（Pexels / Pixabay）

来源：SKILL.md 素材网站采集流程

### 核心原则：实体场景词优先

素材网站搜索时，**搜物理地点/真实环境，不搜抽象技术概念**。

| ❌ 抽象词（返回CG动画） | ✅ 实拍词（返回真实场景） |
|------------------------|------------------------|
| artificial intelligence technology | server room |
| data center blue lights | data center aerial |
| circuit board technology | electronics factory |
| digital network technology | computer lab office |
| AI robot | robot working factory |
| machine learning | automation factory robot |
| future technology | robot assembly line |
| cyber security digital | tech laboratory equipment |

### 关键词语言转换

分镜方案中的关键词是**中文**，素材网站搜索时需要转换为**英文**：

| 分镜方案中文关键词 | 素材网站英文搜索词 |
|-------------------|------------------|
| 半导体 工厂 航拍 | semiconductor factory aerial |
| 芯片 制造 产线 | chip manufacturing production line |
| 光纤 生产 拉丝 | fiber optic production drawing |
| 数据中心 机房 | data center server room |

### 后缀"素材"备选

当英文搜索结果不理想时，可尝试中文关键词加"素材"后缀：

- "半导体素材" → 部分素材网站对中文"素材"后缀有更好匹配
- "芯片制造素材"
- "光纤素材"

### 备选关键词层级

```
第一优先：实体场景英文词（robot working factory）
第二优先：中文关键词加"素材"后缀（半导体素材）
第三优先：更宽泛的英文场景词（automation factory）
```

## 六、地域/文化匹配规则（重要）

### 核心原则：素材来源平台与关键词内容的地域一致

当关键词涉及**国外企业、人物、场景**时，优先去**国外平台**（YouTube / Pexels / Pixabay）搜索，因为国内平台（抖音/B站）上的相关素材少、质量低。

| 关键词类型 | 示例 | 优先平台 | 原因 |
|-----------|------|---------|------|
| 国外企业/品牌 | 英伟达(NVIDIA)、台积电(TSMC)、阿斯麦(ASML) | YouTube → Pexels → B站 | 国外平台有大量工厂实拍、发布会、产线画面 |
| 国外人物 | 黄仁勋、马斯克、山姆·奥特曼 | YouTube（优先） | 国外人物素材集中在YouTube，国内平台极少 |
| 国外场景 | 硅谷、华尔街、东京银座 | YouTube / Pexels | 国外场景实拍素材国外平台更丰富 |
| 国内企业/场景 | 比亚迪、宁德时代、华为 | 抖音 → B站 → YouTube | 国内平台素材更丰富、更匹配 |
| 通用科技/工业 | 芯片制造、光纤、数据中心 | 按实际内容判断 | 国内外平台均可，优先搜到为止 |

### 搜索顺序调整

**国外关键词**（国外企业/人物/场景）：
```
YouTube（优先）→ Pexels/Pixabay → B站 → 抖音
```

**国内关键词**（国内企业/人物/场景）：
```
抖音（优先）→ B站 → YouTube → Pexels/Pixabay
```

**通用关键词**（不分地域的科技/工业/自然类）：
```
抖音 → B站 → YouTube → Pexels/Pixabay/Mixkit
```

### 关键词语言匹配

| 关键词地域 | 平台实拍搜索语言 | 素材网站搜索语言 |
|-----------|----------------|-----------------|
| 国外内容（英伟达、台积电、黄仁勋） | 中文（如"英伟达 工厂"） | 英文（如"NVIDIA factory"） |
| 国内内容（比亚迪、宁德时代） | 中文 | 中文（如"比亚迪 工厂"）或英文 |
| 通用内容（芯片制造、光纤） | 中文 | 英文（优先）/ 中文（备选） |

## 七、规则优先级

当多个规则同时适用时：

1. **分镜方案中的关键词**为第一优先级（人工填写，针对性最强）
2. 如果搜不到 → 切换**备选关键词**（分镜方案中的备选方案列）
3. 如果备选也搜不到 → 尝试**后缀"素材"**（中文）
4. 如果素材网站英文搜索不理想 → 尝试**实体场景词**（英文）
5. 如果实体场景词也不理想 → 尝试**更宽泛的场景词**
