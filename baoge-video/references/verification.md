# 素材验收与审查机制

## 分步审查机制（每个Phase完成后强制执行）

每个Phase完成后，**必须截屏展示给用户检查**，用户确认通过后才进入下一个Phase：

### 审查流程

```
Phase完成 → 截屏/截帧展示 → 用户检查 → 用户确认通过
  → 查阅SKILL.md下一个Phase的要求 → 进入下一个Phase
```

### 各Phase审查内容

| Phase | 审查方式 | 用户检查项 |
|-------|---------|-----------|
| Phase 0 分镜方案 | 表格展示 | 分镜数、时长、素材类型、关键词是否合理 |
| Phase 1 TTS | 播放音频或时长列表 | 音质、语速、时长是否匹配 |
| Phase 2 素材采集 | 截帧对比图（4帧网格） | 6项验收+用户选用确认 |
| Phase 3 贴片制作 | 截屏top/bottom/cap图片 | 暗红底、斜线纹理、金色标题、白字文案 |
| Phase 4 分镜合成 | 每段截1帧展示 | 四层结构、素材区内容、文案条文字、贴片颜色 |
| Phase 5 全帧验证 | 关键帧展示 | 逐项验证结果 |

### ⚠️ 禁止跳过审查

- **禁止**完成一个Phase后直接进入下一个Phase而不展示给用户
- **禁止**仅在文字描述结果而不截屏/截帧展示实际画面
- **禁止**在用户未确认通过时继续推进
- 每次进入新Phase前**必须重新查阅对应章节**，避免遗漏规格要求

## 素材验收12项检查（每段素材下载/生成后强制执行）

用FFmpeg按1秒间隔截帧，**全帧验证**（不是抽帧），每帧逐项检查：

| # | 检查项 | 合格标准 | 不合格处理 |
|---|--------|---------|-----------|
| 1 | 画面方向 | 裁切后能填满素材区1080×768 | 丢弃换源 |
| 2 | 画面内容相关性 | 与文案意象直接相关 | 丢弃换源 |
| 3 | 动态视频 | 有物体移动/镜头移动/场景变化 | 丢弃换源 |
| 4 | 无水印/无logo | 无任何水印、logo、台标、UI弹窗 | 裁剪或模糊处理；无法处理则丢弃 |
| 5 | **无字幕/无文字/无字体** | 画面无任何字幕、文字、弹幕、印刷字体（零容忍） | 丢弃换源 |
| 6 | **无人脸** | 无UP主口播、无博主大头、无半身半脸（零容忍） | 丢弃换源 |
| 7 | 画面审美 | 构图完整、主体居中、清晰不模糊、不丑 | 丢弃换源 |
| 8 | 分辨率 | 裁剪后有效区域≥720×768px | 丢弃换源 |
| 9 | 时长 | 可截取≥6秒有效片段 | 丢弃换源 |
| 10 | **无K线图截取** | 不能截取真实K线图/走势图 | 丢弃换源 |
| 11 | **无PPT/无录屏** | 不能截取PPT/手机录屏/电脑录屏 | 丢弃换源 |
| 12 | **无无关画面** | 必须与当前分镜文案直接相关 | 丢弃换源 |

**任何一项不合格 → 整段丢弃，不裁剪凑合，不模糊凑合。**

### 素材反馈机制（候选素材截帧对比图）

每段平台实拍素材在验收前，**必须生成候选截帧对比图供用户确认**，不能仅文字描述：

1. 每个候选视频用FFmpeg截取4帧（间隔均匀），拼成2×2网格图
2. 网格图上标注：平台、视频ID、每帧时间点
3. 同时对所有候选视频的截帧图并列展示
4. 用 `image` 工具逐张分析，附上6项判断结果
5. 将分析结果和截帧图一起呈现给用户，由用户确认选用哪个候选
6. 用户确认后才进入裁剪清洗和最终验收

## 素材备注清单（每段素材必须记录，不可事后补记）

每段素材下载/生成后**立即**追加一行：

| 字段 | 说明 |
|------|------|
| 分镜编号 | seg00 ~ segNN |
| 时间戳 | 该段在最终视频中的起止时间 |
| 文案内容 | 该段对应的完整文案句 |
| 素材类型 | 实景/AI生成/数据可视化/用户喂素材 |
| 来源平台 | B站/YouTube/抖音/Pexels/AI生成/自研动画 |
| 来源URL | 原始视频完整链接 |
| 搜索关键词 | 搜到这段素材用的关键词 |
| 原始时长 | 源视频完整时长 |
| 截取片段 | 从源视频的什么时间截取了多少秒 |
| 文件名 | 处理后的素材文件名 |
| 验收结果 | 12项检查逐项结果 |
| 处理备注 | 裁剪/模糊/缩放等处理说明 |

AI生成素材额外记录：生成工具、完整prompt、生成次数、动态化方式、为什么用AI。
数据可视化动画额外记录：动画类型、数据来源、生成方式、帧数、动画时长。
