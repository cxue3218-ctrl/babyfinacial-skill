# 数据可视化动画

## ⚠️ 数据可视化触发条件（严格限定，违反即重做）

数据可视化**仅限以下三种情况触发**，其余一律禁止：

| # | 触发条件 | 示例 | 说明 |
|---|---------|------|------|
| T1 | 句子含**具体数值** | "涨价40%"、"192GB"、"从25%到45%" | 有明确数字才用数据可视化 |
| T2 | 句子含**明确比喻词** | "竹竿效应"、"缩圈"、"木桶效应" | 比喻词有对应图形表达 |
| T3 | 句子是**数值对比** | "从25%到45%"、"去年3亿今年8亿" | 两个数值之间的对比 |

**以下情况禁止使用数据可视化**，必须用实景画面或AI生成画面：

| 禁止类型 | 示例 | 正确做法 |
|---------|------|---------|
| 叙述句 | "AI基建趋势行情玩法变了" | 实景/AI画面 |
| 观点句 | "去年是AI必涨" | 实景/AI画面 |
| 判断句 | "今年资金只认业绩验证" | 实景/AI画面 |
| 趋势句 | "资金从芯片转向MLCC" | 实景/AI画面 |

**判断流程**：拿到一句文案 → 是否含具体数值？→ 是否含明确比喻词？→ 是否是数值对比？→ 三个都否 → **禁止数据可视化，用实景或AI画面**。

## ⚠️ 数据可视化铁律（代码级约束，违反即重做）

| # | 铁律 | 违反后果 |
|---|------|---------|
| D1 | **全中文**：画面所有文字必须中文，禁止英文 | 重做 |
| D2 | **禁止纯文字动画**：必须有**图形元素**在动 | 重做 |
| D3 | **图形占比≥70%**：图形动画元素面积≥70%，文字标注≤30% | 重做 |
| D4 | **必须有运动**：柱状图增长、圆圈缩放、箭头移动，不能静态 | 重做 |
| D5 | **禁止HTML静态页**：必须是逐帧动画 | 重做 |
| D6 | **配色统一（禁绿色）**：暗红渐变+斜线纹理，金色标题，钢蓝正向，橙红负向，白字。**全局禁止绿色** | 重做 |
| D7 | **分辨率1080x768**：匹配素材区 | 重做 |
| D8 | **时长匹配TTS**：动画时长=该镜TTS时长，误差≤0.5秒 | 重做 |

## 概念比喻可视化要求

| 文案概念 | 正确MG可视化 | 错误做法（禁止） |
|----------|------------|----------|
| "竹竿效应" | 画竹子/柱状递增图，一根vs多根高度对比动画 | 通用柱状图 |
| "木桶效应" | 画木桶/容器图，最短板限制水位动画 | 无关图 |
| "缩圈" | 画同心圆收缩动画（圆圈逐渐缩小+箭头指向中心） | 纯文字"narrowing" |
| "缺口" | 画缺口/瓶颈示意图，资金箭头涌入动画 | 无关图 |
| "玩法变了" | 左右对比动画（旧规则→新规则，带图形过渡） | 纯文字淡入淡出 |

## 动画类型（必须含图形运动）

| 类型 | 必须的图形动画 | 禁止的做法 |
|------|------------|----------|
| 柱状图对比 | 柱子从0增长到目标高度，颜色区分涨跌 | 静态柱状图 |
| 环形图/饼图 | 扇形逐块展开，百分比数字递增 | 静态饼图 |
| 数字递增 | 数字从0滚动到目标值+背景图形动画 | 纯文字数字 |
| 同心圆缩圈 | 外圆→内圆缩放动画+箭头收缩 | 纯文字"缩圈" |
| 对比过渡 | 左侧图形→右侧图形带移动/变形动画 | 纯文字左右排列 |
| 产业链流程 | 节点逐一亮起+连线动画 | 静态流程图 |
