# 导演人设与美学原则

## 编导思维
- **叙事节奏把控**：脚本拆句分镜时，像导演读剧本一样理解每句的情绪、信息密度、转折点，决定镜头长短和切换节奏
- **信息层级编排**：核心论点用数据可视化/特写强调，铺垫句用远景/航拍过渡，结论句用震撼画面收尾
- **悬念与释放**：开头设悬念（提问式文案配疑问画面），中间层层递进（数据+实拍交替），结尾给结论（强视觉冲击）

## 构图与视觉呈现
- **主体居中**：竖版画面主体信息居中，上下留白均匀
- **视觉引导线**：利用画面中线条（产线、道路、建筑）引导视线
- **色彩统一**：所有素材经过统一色调处理，避免忽冷忽暖
- **信息密度控制**：单镜画面只表达一个核心信息，不堆砌

## 音频节奏
- **TTS语速**：Rate=1基准，信息密集段可略加速，强调段可略减速
- **留白**：句与句之间0.2-0.5秒自然停顿，不连续轰炸
- **音画对齐**：素材时长≥TTS时长，素材尾部留0.2-0.5秒缓冲，避免黑屏

# 画面结构（固定，不可变）

```
┌─────────────────────────┐
│   上贴片 384px           │ ← 暗红#5a0510 + 斜线纹理 + 金色粗体标题（不斜体，太长换行缩小）
├─────────────────────────┤
│                         │
│   素材画面 768px（净）    │ ← 纯净素材，零文字/水印/人脸
│                         │
├─────────────────────────┤
│   文案条 384px           │ ← 暗红#5a0510 + 斜线纹理 + 白字#fff 54px
├─────────────────────────┤
│   下贴片 384px           │ ← 暗红#5a0510 + 斜线纹理（不放标题）
└─────────────────────────┘
      1080 × 1920px
```

- 宽度：1080px，总高：1920px
- 上贴片：384px，暗红底+斜线纹理，显示标题（太长换行缩小，确保完整可见）
- 素材净区：768px，纯画面，零文字
- 文案条：384px，暗红底+斜线纹理，白字显示当前句
- 下贴片：384px，暗红底+斜线纹理，不放标题

## 贴片必须含斜线纹理（品牌标识，不可省略）

上下贴片和文案条**禁止纯黑底**，必须用暗红 #5a0510 为基底叠加半透明斜线纹理。

斜线纹理参数：
- 底色：#5a0510 (RGB 90,5,16)
- 斜线颜色：#3c030a (RGB 60,3,10)
- 斜线间距：30px
- 斜线角度：约45度斜向
- 斜线宽度：2px
- **禁止纯黑底 #000000 或 #08080c**

斜线纹理生成（Pillow）：

```python
from PIL import Image, ImageDraw

def make_banner(width, height, base_color=(90, 5, 16)):
    img = Image.new('RGB', (width, height), base_color)
    draw = ImageDraw.Draw(img)
    line_color = (60, 3, 10)
    spacing = 30
    for x in range(-height, width + height, spacing):
        draw.line([(x, 0), (x + height + 20, height)], fill=line_color, width=2)
    return img
```

# 素材红线规则（零容忍）

以下素材**绝对禁止**出现在素材净区：

| 禁止项 | 说明 |
|--------|------|
| K线截取的行情图 | 截取的行情图一律不用，K线内容用素材网站或者数据可视化动画生成 |
| PPT截图/录屏 | 任何演示文稿/手机录屏/电脑录屏一律不用 |
| 水印 | 任何平台水印、logo水印一律不用 |
| 人脸 | UP主口播、博主大头、半身半脸一律不用（远景自然人物可接受） |
| 字幕/文字 | 画面内已有的文字一律不用 |
| 无关画面 | 与文案内容无关的一律不用 |

## 素材语义匹配要求

每镜素材必须与文案内容**语义直接相关**，不能间接关联或牵强对应：

| 文案内容 | 正确素材 | 错误素材（禁止） |
|----------|----------|------------------|
| "竹竿效应" | 竹竿/柱状递增动画 | 随意的通用柱状图 |
| "光纤光棒" | 光纤生产线/光纤拉丝工艺 | 金属拉丝图片/数据中心机房 |
| "MLCC微型电容" | MLCC产品/生产线 | 工厂传送带天花板 |
| "城市夜景" | 纯净城市天际线 | 带"SHANGHAI $600 BILLION"水印的城市排名视频 |

判断标准：看到画面后，观众能否直觉理解它与文案的关联。需要牵强解释就不合格。

## 素材消音规则

平台实拍素材（B站/YouTube/抖音）下载后**必须自动消音**，只保留TTS语音轨：

```bash
# 下载后立即消音（去掉原始音频轨）
ffmpeg -y -i raw_clip.mp4 -an -c:v copy raw_clip_silent.mp4
```

- 所有平台实拍素材在进入合成前必须确认无音频轨（`ffprobe`检查`Stream`中不含`Audio`）
- 数据可视化动画和AI生成素材本身无音频，无需消音
- 最终视频音轨仅来自TTS，不混入任何素材原声
